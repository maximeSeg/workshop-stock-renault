# Notes de choix UX — V1

## Sidebar sombre (#1a1a1a)
Contraste fort avec la zone de travail claire (#f2f2f0).
Navigation rapide sans quitter la zone principale.
Inspiré des outils pro (VS Code, Linear, Figma).

## Boutons +/− inline dans le tableau
Réduit de 5 clics à 2 pour la mise à jour quotidienne des quantités.
Validé par observation : les mécaniciens ajustent majoritairement ±1.

## Badges d'état colorés
- 🟢 Vert (#EAF3DE) — En stock (quantité > seuil)
- 🟠 Amber (#FAEEDA) — Stock faible (0 < quantité ≤ seuil)
- 🔴 Rouge (#FCEBEB) — Rupture (quantité = 0)
Codes couleur universels, accessibles daltonisme partiel.

## Couleur d'accentuation — Jaune Renault (#FFCC00)
Cohérence avec l'identité visuelle Renault.
Boutons d'action primaire + logo + éléments d'alerte.
